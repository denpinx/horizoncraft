# 基于Godot和C#开发的类Minecraft2D游戏《HorizonCraft》

---

项目已经升级到 godot 4.7

目前已添加55种方块，80种物品,基本上都有对应的贴图。


## 目前支持的功能

- 无限地图
- 死亡重生
- 白昼变化
- 方块光照
- 物品合成
- 结构生成
- 基于组件和系统驱动的方块
- 生物群系
- 物品使用
- 多人联机
- 纯服务端导出

## 备注

- F2 创造模式/生存模式切换
- F3 调试信息
- F4 查看区块信息

## 环境要求

- Godot Engine 4.7 (.NET)
- .NET SDK 8.0

## 运行环境

- Windows 10/11

## 运行项目

- 下载并解压该项目（或使用git clone）
- 运行Godot编辑器，打开该项目
- 点击编辑右上角的运行
