# 树的注册与使用

- 打开[StructManage](..script/WorldControl/Struct/StructManage.cs)
- 在RegStructs静态函数中追加写入以下代码:

```C#
Register(new TreeBuilder("tree_name", "oak_log", "oak_leaves")
{
    RootDeviate = 2,
    RootHigh = new Vector2I(6, 12),
    PeakDeviate = new Vector2I(2, 3),
    PeakHigh = new Vector2I(2, 3),
    LeavesExtend = 0.4f,
});
```
其中"tree_name"是树的自定义id，"oak_log"是注册过的方块，属于树的主干方块，"oak_leaves"是注册二过的方块中的树的树叶方块。

再在地形构造器中(继承了"LandBiome"的类型)的”GeneratorStruct“函数，控制其满足一定条件后，调用"GetStruct"，列:
```C#
BlockStruct blockStrcut =
BlockStructManager.GetStruct("tree_name", lbsc.GlobalX, lbsc.GlobalY, lbsc.GloablZ, lbsc.Random);
lbsc.BlockStructs.Add(blockStrcut);
```
用于获取树的完整结构，并将其添加进结构集合。
