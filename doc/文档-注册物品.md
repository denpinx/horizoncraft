# 注册物品

#### 详细请查看[Json Schema](../config/schema/item.json)

## 打开“config\item\”目录,往里面添加任意文件，文件名不限,但是最好还是根据物品类型进行分类在不同的文件夹中比较好

## 列-注册物品与工具:
```Json
{
  "iron_ingot": {},
  "copper_ingot": {},
  "wood_pickaxe": {
    "max": 1,
    "tags": {
      "fuel": "10",
      "thesaurus": "wood_pickaxe"
    },
    "components": {
      "ToolComponent": {
        "Drive": "ToolSystem",
        "Tag": ["pickaxe"],
        "ToolLevel": 0,
        "Efficiency": 1,
        "Max": 100,
        "Value": 100
      }
    }
  }
}
```
# 解析

- 绝大多数字段可以不写,会有默认值
- 物品可以多个组件,但是同一个组件类只能有一个，不能完全重复，但可以是派生的子类或父类。
- 物品配置的优先级 > 方块配置,会覆盖方块的配置，方块会默认自动生成对应的物品配置(如果没有被提前加载的话)
