# 语言文本的注册和使用

#### 详细请查看[Json Schema](../config/schema/lang.json)

## 注册
### 在config/lang目录下创建任意json文件

```Json
{  
  "lang": "cn",
  "prefix": "meta",
  "texts": {
    "my_item": "我的物品"
  }
}
```

```Json
{  
  "lang": "cn",
  "prefix": "ui",
  "texts": {
    "my_item_show_tag": "物品 {0},{1}"
  }
}
```

### 解析
    lang 属性表示当前文本属于什么语言
    prefix,在注册时会自动添加,默认可以不用配置

## 使用语言文本

- 动态创建的文本可以直接调用 "my_item".Trprefix("meta") 或 "my_item_show_tag".Trprefix("meta",a,b)支持字符格式化
- "xxx".Tr(params string[] args)只支持字符串格式化
- "xxx".Trprefix(string prefix,params string[] args)支持前缀和字符串格式化 
- 静态文本初始化需要继承ITranslatable接口,实现TranslateChange方法，只能在实现TranslateChange方法里实现初始化静态文本的翻译
- 切换语言时会遍历场景树上所有实现了ITranslatable接口的节点，并调用TranslateChange方法。
