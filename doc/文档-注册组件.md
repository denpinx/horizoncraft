# 注册组件

### 1.新建组件类,继承[组件](../script/Components/Component.cs),注册新的组件类型

```C#
    [MemoryPackable]
    public partial class MyComponent : TickComponent
    {
        //定义字段
    }
```

### 2.打开 [组件系统枚举](script/Components/SystemEnum.cs),添加新的组件枚举

### 3.创建 System对象 System有物品,实体,组件之分,根据需求继承。

```C#
    //组件系统
    public class MyItemSystem:ItemComponentSystem
    {
        //在内部重载事件即可
    }
    //方块系统
    public class MyBlockSystem:TickSystem
    {
        //在内部重载事件即可
    }
```

### 4.打开[组件管理器](script/Components/ComponentManager.cs),注册新的组件类型

```C#
    static ComponentManager()
    {
        ...
        //在尾部添加
        Register(SystemEnum.MyComponentSystem, typeof(MyComponent), new MySystem());
    }
```

### 5.在[组件基类](../script/Components/Component.cs)类中标记新建的组件,用于MemoryPack序列化

```C#
    //在这里添加，注意下标请勿重复,下标id要延续上一个id自增。
    [MemoryPackUnion(16, typeof(MyComponent))]
    public abstract partial class Component
    {
        ....
    }
```
