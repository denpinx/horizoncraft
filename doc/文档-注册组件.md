# 注册组件

### 1.新建组件类,继承[组件](../script/Components/Component.cs),注册新的组件类型

```C#
    [MemoryPackable]
    public partial class MyComponent : TickComponent
    {
        //定义字段
    }
```

### 2.打开[组件管理器](script/Components/ComponentManager.cs),注册新的组件类型

```C#
    static ComponentManager()
    {
        ...
        //在尾部添加
        Register("组件行为", () => new MyComponent(), new TickSystem()
        {
            Tick = (BlockTickEvent e, TickComponent cmp) =>
            {

            }
        });
    }
```

### 3.在[组件](../script/Components/Component.cs)类中标记新建的组件,用于MemoryPack序列化

```C#
    //在这里添加，注意下标请勿重复
    [MemoryPackUnion(999, typeof(MyComponent))]
    public abstract partial class Component
    {
        public string Name;
    }
```
